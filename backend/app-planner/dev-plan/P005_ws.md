# P005 WebSockets

Goal: add authenticated realtime transport.

Do:
- bootstrap the websocket server and gateway
- authenticate connections and scope subscriptions by tenant, facility, role, and module
- add shared emit helpers so services can publish domain events without transport leakage

Acceptance:
- websocket auth matches backend access rules
- event payloads are stable and `snake_case`
- services can publish realtime updates without bypassing module boundaries
