# hms-flutter-expressjs

Hospital management system codebase with a Flutter frontend and an Express.js backend.

## Structure

- `frontend/` - Flutter application
- `backend/` - Express.js API and Prisma database layer
- `app-planner/` - planning and project notes

## Web development

Open this repository in VS Code and run
`Frontend: Flutter Web Chrome (development)` from **Run and Debug**. The
frontend launches at `http://127.0.0.1:5201` and the workspace triggers Flutter
hot reload whenever files are saved during an active debug session.
